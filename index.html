Open Source at Github
Podcast-me é um projeto para comemorar o dia 21 de outubro, que é o Dia do Podcast e é dedicado a todos os ouvintes, podcasters e simpatizantes. Participe!
Esse software utiliza as seguintes bibliotecas:
jQuery
jQuery UI
Bootstrap
Font Awesome
jQuery UI Touch Punch by David Furfero
Simple Colorpicker by Tanguy Krotoff
File Saver by Eli Grey
Canvas toBlob by Eli Grey and Devin Samarin
Icon made by Freepik from www.flaticon.com is licensed under CC BY 3.0

Privacidade

Esse software não envia a imagem carregada para processamento em um servidor remoto. Toda a carga, processamento e geração da imagem final é feita localmente através de scripts que rodam no próprio navegador.
